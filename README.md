# PRTMCoordTrans

PRTMCoordTrans is a simple library which transforms WGS84 latitude/longitude coordinates into katech TM coordinates and reverse. 


## Installation

- Drag the `PRTMCoordTrans` folder into your project.
- `#import PRTMCoordTrans.h`
